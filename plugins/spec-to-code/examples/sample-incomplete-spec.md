# 기획서: 장바구니 쿠폰 적용

## 개요
사용자가 장바구니에서 쿠폰 코드를 입력하면 할인이 적용된다.

## 화면
- 장바구니 페이지 하단에 "쿠폰 코드" 입력칸과 "적용" 버튼이 있다.
- 쿠폰을 적용하면 할인 금액이 표시되고 최종 결제 금액이 줄어든다.

## 동작
1. 사용자가 쿠폰 코드를 입력하고 "적용" 을 누른다.
2. 유효한 코드면 할인이 적용되고, 할인액과 최종 금액을 보여준다.
3. 쿠폰은 정액(예: 5,000원 할인) 또는 정률(예: 10% 할인) 두 종류가 있다.

## 비고
- 쿠폰 코드는 백엔드 `POST /api/coupons/validate` 로 검증한다.
